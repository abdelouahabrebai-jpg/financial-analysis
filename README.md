# Financial Market Analysis

Analyse de marchés financiers en Python — projet pédagogique.

## Actifs analysés
- Apple (AAPL)
- Bitcoin (BTC-USD)
- S&P 500 (^GSPC)
- Or (GC=F)
- Pétrole WTI (CL=F)

## Ce que le projet couvre
- Récupération de données avec yfinance
- Calcul des rendements logarithmiques
- Volatilité et ratio de Sharpe
- Matrice de corrélation
- Analyse contextuelle 2021-2026

## Bibliothèques
yfinance, pandas, numpy, matplotlib, seaborn
